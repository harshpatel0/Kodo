# Custom Instructions

Use this file to specify any custom instructions for Kodo e.g.,

- Be as brief as possible
- Use skills
- Take the least path of resistance to complete a given task

Specify your default browsers, and apps you use to complete tasks
If this relates to the `browser-navigation` skill, the skill by default will open your selected settings browser

> Once you are done with editing this file, rename the file to `custom_instructions.md` to allow Kodo to pick it up and add it to the system prompt.
